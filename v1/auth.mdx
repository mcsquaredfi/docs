    ## Auth

    ### Login

    - Code: 10020

    #### Parameters

    | Name            | Type              |
    |-----------------|-------------------|
    | address         | BlockchainAddress |
    | signature_text  | String            |
    | signature       | String            |
    | service         | EnumRef: service  |
    | device_id       | String            |
    | device_os       | String            |

    #### Returns

    | Name            | Type              |
    |-----------------|-------------------|
    | address         | BlockchainAddress |
    | display_name    | String            |
    | avatar          | Optional: String  |
    | role            | EnumRef: role     |
    | user_id         | BigInt            |
    | user_token      | UUID              |
    | admin_token     | UUID              |
